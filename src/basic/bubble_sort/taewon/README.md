## 계산법

```
10 + 9 + 8 + .. + 1 

=> 10 * (10 + 1) / 2 = 55 등차수열
=> N * (N + 1) / 2
=> N * N, 결과에 큰 영향을 끼치지 않는 +, -, / 등은 계산에서 생략한다
=> O(N * N) Big O 표기법
```

시간 복잡도는 O(N^2)
